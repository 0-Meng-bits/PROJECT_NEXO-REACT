Conceptual Framework

NEXO Connect — Interest-Based Student Community Platform · CTU–Daanbantayan

NEXO CONNECT
A verified, interest-driven digital campus community platform that bridges students,
faculty, and organizations of Cebu Technological University – Daanbantayan Campus.
Student Engagement Digital Identity Community Building Real-Time Communication Academic Integration
📥 Input
User Data
Student / Faculty Registration
CTU ID, full name, email, password, user type (Student / Faculty / Admin)
School ID Photo (OCR)
Uploaded ID image scanned via Tesseract.js to extract and verify CTU ID number
Onboarding Profile
Profile photo, course, year level, and interest selections (22 interest categories)
Community Data
Circle Creation
Circle name, category (Academic / Project / Hobby / Social), icon, description
Audition Forms
Custom questions created by circle leaders for membership screening
Content Data
Messages & Announcements
Real-time chat messages, campus-wide posts, circle announcements, reactions
Campus Events
CTU AY 2025-2026 academic calendar: semester dates, exams, enrollment, holidays
⚙️ Process
Authentication & Verification
Identity Verification
OCR scans uploaded school ID → extracts CTU ID → admin reviews and approves account
JWT Session Management
Supabase Auth issues JWT tokens; protected routes validate session on every page load
Matching & Personalization
Aura Match Algorithm
Counts students sharing selected interests; maps interests to circle categories; suggests relevant circles during onboarding
Interest-Based Discovery
Explore page filters circles by category; featured circles surface based on activity
Community Management
Membership & Rank System
Join requests → Leader approval → Member / Moderator / Co-Leader / Leader hierarchy with caps
Audition System
Leaders build custom forms → applicants submit answers → Phase 1 review → optional Phase 2 live screening
Real-Time Engine
Supabase Realtime
WebSocket subscriptions push new messages, reactions, notifications, and announcements instantly to all connected users
📤 Output
For Students
Verified Digital Identity
Authenticated CTU student profile with course, year level, interests, and profile photo
Circle Membership
Access to joined circles, channels, announcements, and ranked roles within communities
Real-Time Communication
Live chat with message reactions (❤️😂😢), edit, delete, and view member profiles
Campus Awareness
Home feed with announcements, campus events calendar, and notifications
For Faculty / Admins
Admin Dashboard
Verification queue, analytics charts, user management, circle oversight, campus feed moderation
Campus Event Management
Post and manage CTU events visible to all students on the color-coded academic calendar
System Outcome
Connected Campus Community
A unified, safe, and verified digital space that reduces campus disconnection and increases student engagement
🔬 Theoretical Grounding
  • Social Capital Theory — digital circles build bonding and bridging capital among CTU students
  • Community of Practice (Wenger) — shared interest groups foster collaborative learning and identity
  • Uses & Gratifications Theory — students use the platform to fulfill social, informational, and entertainment needs
  • Self-Determination Theory — interest-based matching supports autonomy, competence, and relatedness
🎯 Expected Outcomes
  • Increased student participation in campus organizations and activities
  • Reduced information gap between students, faculty, and administration
  • Streamlined organization membership through the audition system
  • Stronger sense of campus identity and belonging among CTU students
  • Centralized, verified, and real-time campus communication channel
